# Чеклист тестирования ToDoDesk

Перед каждым релизом пройти на portable-сборке (`npm run dist`).

## Запуск и данные

- [ ] Portable запускается, данные создаются рядом с `.exe` (`data.tododesk`)
- [ ] Dev-режим (`npm run dev:clean`) не конфликтует с portable из трея
- [ ] Автобэкап создаёт файлы в `backups/`
- [ ] Импорт: замена / объединение / новый проект — без потери полей 1.4
- [ ] Экспорт и повторный импорт — данные совпадают
- [ ] Шифрование: включить → перезапуск → ввод пароля → данные на месте
- [ ] Ошибка пароля не перезаписывает файл пустым

## Задачи

- [ ] Создание в Сегодня / Входящие / проект / тег (автотег)
- [ ] Подзадачи, чеклист, теги, вложения
- [ ] Markdown в описании отображается
- [ ] Повтор: день, неделя, месяц (в т.ч. 31 янв), будни, выходные
- [ ] Зависимости: цикл блокируется
- [ ] Удаление задачи чистит доску, черновики, вложения
- [ ] Очистка выполненных — то же
- [ ] Кастомное напоминание сохраняется при смене срока
- [ ] Ctrl+Z отменяет последнее действие

## Виды

- [ ] Сегодня, Inbox, All, Completed, календарь, таймлайн, спринты
- [ ] Kanban: поиск в обеих колонках, drag между колонками
- [ ] Доска: фильтр, поиск блоков, снимки, откат, PNG
- [ ] Статистика, заметки, фокус

## Синхронизация

- [ ] Watcher при смене файла в sync-папке
- [ ] «Отправить сейчас» / «Получить сейчас» в настройках
- [ ] Конфликт при несохранённых правках в TaskDetail
- [ ] Заголовок окна `*` при несохранённых правках

## UI и хоткеи

- [ ] Ctrl+K палитра, Ctrl+F поиск, Ctrl+Shift+V вставка
- [ ] Ctrl+1–7 переключение видов
- [ ] Умные правила: вкл/выкл в настройках, применение после перезапуска
- [ ] Темы, акцент, компактный сайдбар
- [ ] Уведомления (срок, dueTime, тихие часы)

## Сборка

- [ ] `npm run build` — OK
- [ ] `npm run typecheck` — OK
- [ ] `npm test` — OK
- [ ] `npm run dist` — portable exe создаётся
