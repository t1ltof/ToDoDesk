## ToDoDesk 0.8.1

**Рекомендуемая версия.** Патч-релиз: исправления по результатам аудита кода и тестирования.

Десктопный планировщик задач для Windows (portable).

### Целостность данных

- Merge/import сохраняют все поля формата 1.4: вложения, спринты, снимки доски, умные правила, черновики, история доски
- Импорт и экспорт поддерживают зашифрованные `.tododesk` файлы
- При ошибке расшифровки данные **не перезаписываются** — открываются настройки для ввода пароля
- Запись на диск блокируется, если включено шифрование, но пароль не введён

### Синхронизация и несохранённые правки

- Автосохранение больше не сбрасывает флаг несохранённых правок в TaskDetail
- Исправлен баг, из-за которого фокус окна сбрасывал статус «есть несохранённые изменения»
- Sync watcher перезапускается только при смене папки синхронизации

### Безопасность

- Валидация путей вложений (`tododesk-attachment://`) — защита от path traversal

### Задачи и напоминания

- Удаление задач / очистка выполненных — также чистит узлы доски, черновики, вложения
- Кастомные напоминания не удаляются при смене срока задачи
- Учёт `dueTime` в уведомлениях
- Проверка циклов зависимостей `dependsOnTaskId`
- Исправлен monthly recurrence (31 янв → последний день месяца)

### UI

- Умные правила применяются без автозаписи на диск
- Автотег при создании задачи из вида `tag:*`
- Поиск в Kanban-колонке «Выполнено»
- Сброс bulk-режима при смене вида

### Установка

Скачайте `ToDoDesk-0.8.1-portable.exe` и запустите. Данные сохраняются рядом с файлом в `data.tododesk`.

### Формат данных

Версия схемы **1.4**.
