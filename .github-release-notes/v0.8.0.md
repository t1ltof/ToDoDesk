## ToDoDesk 0.8.0

Десктопный планировщик задач для Windows (portable).

> Доступен патч [v0.8.1](https://github.com/t1ltof/ToDoDesk/releases/tag/v0.8.1) с исправлениями целостности данных и синхронизации.

### Новое

**Скорость работы**
- Командная палитра (**Ctrl+K**)
- Вставка списка задач из буфера (**Ctrl+Shift+V**)
- Недавние задачи в сайдбаре

**Доска**
- Рамка выделения (Shift — добавить к выбору)
- Выровнять / Сетка для выбранных блоков
- Превью на блоках: срок, теги, чеклист, SLA, иконки статуса
- Подзадачи на блоках
- Снимки раскладки и откат доски
- Экспорт доски в PNG

**Задачи**
- Markdown в описании
- Вложения файлов (drag-drop)
- Повтор по будням / выходным
- Умные правила (просрочка → Входящие)
- Создать проект из ветки задач
- Автосохранение черновиков

**Новые виды**
- Таймлайн (30 дней)
- Спринты
- Следующая задача
- Еженедельный обзор (мастер)
- Сравнение недель и прогноз загрузки в статистике

**Надёжность и настройки**
- Экспорт CSV
- Диалог конфликтов синхронизации
- Звук уведомлений и ежедневная сводка
- Акцентный цвет, компактный сайдбар
- Скролл только в левой колонке

### Установка

Скачайте `ToDoDesk-0.8.0-portable.exe` и запустите. Данные сохраняются рядом с файлом в `data.tododesk`.

### Формат данных

Версия схемы **1.4** (миграция из 1.0–1.3).
