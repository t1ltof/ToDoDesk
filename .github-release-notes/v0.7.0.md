## ToDoDesk 0.7.0

Десктопный планировщик задач для Windows (portable).

> Рекомендуется установить последнюю версию: [v0.8.1](https://github.com/t1ltof/ToDoDesk/releases/latest)

### Новое

- **Доска задач** — drag-and-drop, связи между блоками, zoom, полноэкранный режим, выбор цвета фона
- **Мультивыделение** — Ctrl + клик, перетаскивание нескольких блоков вместе
- **Задачи** — закрепление, архив, зависимости, массовые действия, быстрые фильтры
- **Глобальный поиск** (Ctrl+F), расширенные хоткеи
- **Календарь** — режим дня, цели недели
- **Статистика** — тепловая карта, экспорт отчёта
- **Заметки**, **Фокус/Pomodoro**, **шаблоны проектов**
- **Темы** оформления (тёмная / AMOLED / светлая)
- Импорт **Объединить**, автобэкап, синхронизация через папку, шифрование, тихие часы

### Установка

Скачайте `ToDoDesk-0.7.0-portable.exe` и запустите. Данные сохраняются рядом с файлом в `data.tododesk`.

### Формат данных

Версия схемы **1.3** (совместима с миграцией из 1.0–1.2).
