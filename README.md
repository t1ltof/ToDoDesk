# ToDoDesk

Десктопный планировщик рабочих задач для личного использования (Windows, portable).

## Возможности (v0.1)

- Проекты и дерево задач
- Виды: Сегодня, Входящие, Все задачи, по проекту
- Приоритет, статус, срок (дата)
- Экспорт и импорт `.tododesk`
- Импорт: заменить всё или как новый проект
- Тёмная тема, русский интерфейс
- Уведомления Windows о просроченных и сегодняшних задачах
- Данные в JSON рядом с `.exe` (в dev — папка `data-dev`)

## Разработка

```bash
npm install
npm run dev
```

## Сборка portable

```bash
npm run dist
```

## Стек

- Electron + React + TypeScript
- Tailwind CSS
- Zustand + Zod
- JSON-файл вместо СУБД
