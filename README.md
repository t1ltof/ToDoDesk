# ToDoDesk

Десктопный планировщик рабочих задач для личного использования (Windows, portable).

## Возможности (v0.4)

- Проекты и дерево задач (неограниченная вложенность)
- Панель деталей задачи: описание, срок, приоритет, проект
- Подзадачи, чеклисты, теги
- Виды: Сегодня, Входящие, Все задачи, Выполненные, по проекту
- Очистка выполненных задач
- Редактирование и удаление проектов
- Drag-and-drop сортировка задач
- Системный трей (сворачивание при закрытии окна)
- Глобальный хоткей Ctrl+Shift+T — быстрое добавление задачи
- Portable-сборка для Windows
- Поиск по названию и описанию
- Экспорт и импорт `.tododesk`
- Диалог импорта с превью: заменить всё или как новый проект
- Тёмная тема, русский интерфейс
- Уведомления Windows о просроченных и сегодняшних задачах
- Данные в JSON рядом с `.exe` (в dev — папка `data-dev`)

## Разработка

```bash
npm install
npm run dev
```

## Сборка portable

```bash
npm run dist
```

Готовый файл: `release/ToDoDesk-0.4.0-portable.exe`

Данные (`data.tododesk`) сохраняются **рядом с `.exe`**.

## Горячие клавиши

| Клавиши | Действие |
|---------|----------|
| Ctrl+Shift+T | Открыть окно + быстрое добавление задачи |
