# ToDoDesk

Десктопный планировщик рабочих задач для личного использования (Windows, portable).

**Актуальная версия:** [0.8.1](https://github.com/t1ltof/ToDoDesk/releases/latest) · формат данных **1.4**

## Скачать

[ToDoDesk-0.8.1-portable.exe](https://github.com/t1ltof/ToDoDesk/releases/download/v0.8.1/ToDoDesk-0.8.1-portable.exe) — скачать, запустить, готово.

Данные (`data.tododesk`, бэкапы, вложения) сохраняются **рядом с `.exe`**. В режиме разработки — в папке `data-dev/`.

## Возможности

### Задачи и проекты

- Проекты и дерево задач (неограниченная вложенность)
- Панель деталей: описание (Markdown), срок, время, приоритет, проект, зависимости
- Подзадачи, чеклисты, теги, вложения файлов (drag-and-drop)
- Повторение: день / неделя / месяц / будни / выходные
- Закрепление, архив, массовые действия, быстрые фильтры
- Шаблоны задач и шаблоны проектов
- Создать проект из ветки задач
- Умные правила (просрочка → Входящие / тег)
- Автосохранение черновиков названия и описания
- Спринты, «Следующая задача», еженедельный обзор

### Виды

- Сегодня, Входящие, Все задачи, Выполненные
- По проекту (список и Kanban) и по тегу
- Календарь (неделя и день), цели недели
- Таймлайн (30 дней)
- Доска идей (связи, группы, снимки, история, экспорт PNG)
- Статистика (графики, тепловая карта, сравнение недель, прогноз)
- Заметки, Фокус / Pomodoro

### Скорость работы

- Командная палитра (**Ctrl+K**)
- Глобальный поиск (**Ctrl+F**)
- Вставка списка задач из буфера (**Ctrl+Shift+V**)
- Недавние задачи в сайдбаре
- Drag-and-drop сортировка задач
- Отмена действия (**Ctrl+Z**)

### Надёжность и настройки

- Экспорт и импорт `.tododesk` (замена / объединение / новый проект), поддержка шифрования
- Экспорт CSV и отчёта
- Автобэкап, синхронизация через папку, диалог конфликтов
- Шифрование файла данных паролем
- Проверка обновлений с GitHub
- Уведомления Windows (срок, просрочка, кастомные напоминания, ежедневная сводка)
- Тихие часы, звук уведомлений
- Темы: тёмная / AMOLED / светлая, акцентный цвет, размер шрифта
- Системный трей, счётчик задач на иконке
- Глобальный хоткей **Ctrl+Shift+T** — быстрое добавление из любого места
- Русский интерфейс

## Горячие клавиши

| Клавиши | Действие |
|---------|----------|
| Ctrl+Shift+T | Показать окно + быстрое добавление (глобально) |
| Ctrl+N | Быстрое добавление задачи |
| Ctrl+K | Командная палитра |
| Ctrl+F | Глобальный поиск |
| Ctrl+Shift+V | Вставить список задач из буфера |
| Ctrl+Z | Отменить последнее действие |
| Ctrl+1 … Ctrl+7 | Виды: Сегодня, Входящие, Все, Выполненные, Календарь, Доска, Статистика |
| / | Фокус на поле поиска |
| Escape | Снять выделение задачи |
| Delete | Удалить выбранную задачу |

## Разработка

```bash
npm install
npm run dev
```

Перед `npm run dev` закройте portable-версию (`ToDoDesk.exe` из `release/`), иначе может открыться старая сборка из трея. Для чистого запуска:

```bash
npm run dev:clean
```

Окно разработки имеет заголовок **«ToDoDesk — Разработка»**.

Проверка типов:

```bash
npm run typecheck
```

## Сборка portable

```bash
npm run dist
```

Готовый файл: `release/ToDoDesk-<версия>-portable.exe`

Скрипт обновляет `app.asar` в `release/win-unpacked` и собирает portable без повторной загрузки Electron. При первой сборке на новой машине нужна папка `release/win-unpacked` (скопируйте с предыдущей сборки) или полная упаковка:

```bash
npm run dist:full
```
